# 💰 Project InMyPocket: The Financial Brain for VA

![InMyPocket Mockup](file:///C:/Users/sunwo/.gemini/antigravity/brain/ab93820d-0555-4098-b350-df9d2a9d569d/in_my_pocket_dashboard_mockup_1776354815075.png)

## 1. 프로젝트 비전 (Vision)
**InMyPocket**은 단순한 가계부나 자산 관리 앱이 아닙니다. 이 앱은 **VA_Project** 에이전트가 사용자의 자산을 지키고 성장시키기 위해 사용하는 **"데이터 엔진이자 액션 센터"**입니다.

## 2. 통합 하네스 아키텍처 (Synergy Tracks)

| 프로젝트 | 역할 | 기술 스택 |
|---|---|---|
| **VA_Project** | **Interactive Persona** | Python/Node.js (Metric Learning, Avatar) |
| **InMyPocket** | **Financial Intelligence** | Next.js, Supabase, Stripe, Tailwind |

## 3. 초기 구축 과제 (Current Harness)

| 트랙 | 성격 | 과제 |
|---|---|---|
| **Track A: Data** | **Supabase** | 자산 정보 및 거래 내역 스키마 정의 (`supabase/migrations`) |
| **Track B: UI/UX** | **Vanguard** | `VA_Project` 아바타가 상주하는 프리미엄 대시보드 구현 |
| **Track C: Logic** | **Fin-Agent** | Stripe 테스트 결제 및 수익 분석 로직 스텁 구현 |

## 4. Next Step
1. `InMyPocket`의 Supabase DB 스키마 설계 (`users`, `assets`, `transactions`).
2. `.env.local` 파일에 Supabase/Stripe 테스트 키 스텁 구성.
3. `VA_Project` 아바타를 대시보드 우측 하단에 플로팅 컴포넌트로 배치.
